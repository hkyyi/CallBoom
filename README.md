# CallBoom

## 项目描述

本项目通过采集[莆田系医院网址](api.txt)，然后用[DrissionPage](https://github.com/g1879/DrissionPage)对浏览器模拟作将目标手机号发送给企业，让企业销售联系目标联系人。

## 使用教程

1. 请移步到项目内下载callboom.exe

2. 双击运行callboom.exe，输入手机号码
